함수

변수

* 전역변수 : 모든 스코프에서 사용 가능 한 변수
(함수 밖에서 선언 후 사용, 함수 안에서 사용할 경우 var 타입을 제거해야함)
* 지역변수 : 특정 스코프에서만 사용 가능 한 변수

호이스팅 : '끌어올린다'라는 개념으로 사용됨

자바 스크립트는 인터프리터(순차적 해석)를 사용해서 코드를 해석
함수를 만나면 함수 전체를 스캔
함수에 선언이 존재할 경우 별도로 저장

let 타입

* 호이스팅, 재선언 불가

const 타입

* 호이스팅, 재선언 불가
* 선언 시 초기화 필수
* 선언된 변수 재사용 불가

var 타입

* 호이스팅, 재사용 가능
* 재선언 가능하나 사용 하지 말것
* 어지간하면 쓰지말것, 전역 변수에만 사용 권장
